# Tone & Writing Guide

This document defines the writing tone for all WoWS Translator content — phrase translations, replies, truths, and class dialogues.

## Core Principles

The translator is a **meme tool first, translation tool second**. Every piece of content should feel like it was written by a seasoned WoWS player with a dark sense of humor about the game they love to hate.

## Tone Levels

### 1. Humorous & Self-Deprecating (Primary)

The default tone. Pokes fun at how seriously players take the game while acknowledging that we're all in the same salty boat.

**Example:**
> "我打得很努力但输了，问题一定不出在我身上。"
> → Acknowledges the universal impulse to blame others while gently mocking it.

### 2. Sarcastic & Bitter (Secondary)

For high-salt phrases. Channels the exhausted energy of someone on a 10-game losing streak.

**Example:**
> "WG分配系统是一台精密的人类情绪研究装置。"
> → Bitter but creative. Makes the reader laugh despite themselves.

### 3. Wholesome & Uplifting (Rare)

Used sparingly for the "Truths" feature. These should still carry a hint of shared suffering.

**Example:**
> "一局游戏的胜负在匹配时就已决定，剩下的20分钟只是形式。"
> → Bleak but unifying. We're all in this together.

### 4. Pure Chaos (The Edge)

For class dialogues and special entries. Absurdist humor about the game's inherent ridiculousness.

**Example:**
> "我开了雷达，点亮了DD，然后发现队友全在20公里外。我的牺牲换来了0个助攻。"
> → The punchline is the game itself.

## Content Guidelines

### DO ✅

- Make fun of game mechanics, not individual players
- Use self-deprecating "we" language ("我们") to include yourself
- Reference actual game situations (missing salvos, bad MM, radar timing)
- Keep replies under 30 characters when possible (snappy!)
- Use Chinese internet slang when appropriate and understandable

### DON'T ❌

- Use genuinely insulting or harassing language
- Make personal attacks on specific player groups
- Reference real-world politics or sensitive topics
- Use overly obscure memes that only 1% of players would get
- Write replies that could be used to actually harass other players in chat

## Salt & Blame Score Guide

| Score | Salt Level | Description |
|-------|-----------|-------------|
| 0–20 | Low | Genuine questions, game info, neutral observations |
| 21–40 | Mild | Slight annoyance, dry humor |
| 41–60 | Medium | Clearly frustrated, sarcastic |
| 61–80 | High | Tilted, blaming, passive-aggressive |
| 81–100 | Extreme | Maximum sodium, keyboard-smashing territory |

| Score | Blame Level | Description |
|-------|------------|-------------|
| 0–20 | Self | Taking responsibility, self-deprecating |
| 21–40 | Neutral | Observation without attribution |
| 41–60 | Soft Blame | Implied criticism |
| 61–80 | Hard Blame | Directly pointing at others |
| 81–100 | Pure Deflection | Everything is someone else's fault |

## Language Mix

- **English phrases** → Chinese explanations + Chinese replies (bilingual humor)
- **Chinese phrases** → Stay in Chinese throughout
- **Tags** → Chinese for internal use, English for API compatibility

The bilingual nature of the content is intentional — it reflects the mixed-language reality of Asia-server WoWS chat.
